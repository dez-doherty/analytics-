# Analytics+ (Attendance Dashboard)

## Description

A Chrome extension that injects an interactive analytics dashboard into the Aston University attendance page, visualising attendance patterns directly alongside the existing UI

## Technologies 

- `React`
- `JavaScript`
- `Vite`
- `ApexCharts`
- `Material UI`
- `Gridstack`
- `Chrome Extension API (Manifest V3)`

## Features

- `Reads attendance data straight from the page DOM`
- `Interactive heatmap of attendance across weeks and sessions`
- `Radial chart summarising your overall attendance rate`
- `Drag-and-drop dashboard grid to add and rearrange charts`
- `Styled to match the host attendance page`
